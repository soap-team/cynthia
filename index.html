Cynthia: diff labelling interface

This web app allows you to help the SOAP team (VSTF/GRASP) by labelling and categorising diffs. Labelled data will be used to develop, train and improve our detection tools.

In order to start helping, you must have over 20 reports to the SOAP team, and you will need to obtain a password from Noreplyz.
